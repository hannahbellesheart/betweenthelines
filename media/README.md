# Media Folder

This folder is for storing your MP3 audio files.

## How to Add Your Music

1. Place your MP3 files in this `media` folder
2. Edit the `player.js` file and add your tracks to the playlist array in the `loadPlaylist()` method

Example:
```javascript
this.playlist = [
    { title: 'My First Song', file: 'media/song1.mp3' },
    { title: 'My Second Song', file: 'media/song2.mp3' },
    { title: 'My Third Song', file: 'media/song3.mp3' },
];
```
